# Jamileth Jackelinne Guerra Aguilar

Estratega publicitaria y AI Hybrid con enfoque en expansión de marca, activaciones 360° y experiencias digitales que conectan negocio y creatividad. Combino análisis estratégico, planificación de medios y diseño de flujos para construir campañas coherentes, medibles y listas para optimizarse con datos.

**Especialidades clave**
- Roadmaps estratégicos para marcas que buscan escalar en LATAM.
- Diseño de experiencias y sistemas visuales con narrativa clara.
- Planificación y optimización de medios orgánicos y pagados con mentalidad SEM/SEO.
- Integración de IA en procesos de marketing para prototipado y validación rápida.

**Forma de trabajar**
- Briefs accionables con hipótesis, ventajas competitivas y asignación de medios.
- Flowcharts y journey maps para asegurar coherencia entre canales y momentos de verdad.
- Planes de medios estructurados en cuadros por canal, formato y ventana temporal.
- Panel de control propio para mantener portafolio, ideas y anexos listos para compartir.

Si encontraste este repositorio, tienes acceso a mis casos, fraguas de ideas y al panel donde organizo entregables y programación de contenido. Escríbeme para colaboraciones o consultorías.
